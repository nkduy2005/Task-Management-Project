## Task management (API PROJECT)

## Overview
Task Management API is a RESTful backend system built with Laravel. The system allows users to manage their personal tasks securely with authentication and ownership-based authorization. Each user can create, update, delete, and manage only their own tasks.

## Features
View task lists
Create new task
View task deatails
Update task
Update status task
Delete task

## Teck stack
Laravel 12
MYSQL
Laravel Sanctum
Sramble (API documentation)

## Installation 
- Download zip project or clone project 
- composer install 
- cp .env.expamle .env 
- php artisan key:generate 
- php artisan migrate 
- php artisan db:seed 
- php artisan serve 
- Access http://localhost:8000/docs/api
